# Avance 60% – Proyecto de Grado II

Este repositorio reúne los entregables del primer avance del 60% del proyecto **Modelos sistémicos para el fortalecimiento del cine independiente colombiano**.

## Incluye
- Documento general del proyecto actualizado.
- Documento analítico del **Objetivo Específico 1**.
- Avance preliminar del modelo de dinámica de sistemas en **PowerSim**, basado en **Hughes (2012)**.

## Estado del desarrollo
- **Hughes (2012)**: implementado de forma preliminar.
- **Oruç (2010)**: diseñado teóricamente.
- **Lane & Husemann (2004)**: diseñado teóricamente.

Este avance consolida la base documental, analítica y técnica para la siguiente fase del proyecto.
